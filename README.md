# MIME

A library for that maps mime types to extensions and vice-versa.

## Installation

The package can be installed as:

  1. Add mime to your list of dependencies in `mix.exs`:

        def deps do
          [{:mime, "~> 1.0"}]
        end

  2. Ensure mime is started before your application:

        def application do
          [applications: [:mime]]
        end
